## Ćwiczenie 10
Synchronizacja działania wątków za pomocą algorytmu piekarni.

## Uruchamianie
1. Użyj polecenia `make run`
2. Ilość sekcji krytycznych można zmienić za pomocą zmiennych `PROC_N` i `CRYT_N` w pliku `Makefile`

## Struktura projektu
```
├── Makefile
├── README.md
└── src
    ├── bakery.c
    ├── include
    │   └── bakery.h
    └── main.c
```
